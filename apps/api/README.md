## Commands executed

```bash
# Install Fastify and more
npm i fastify fastify-type-provider-zod @fastify/cors zod

# Install Prisma
npm i prisma -D  

# Inicialize Prisma
npx prisma init  

# Run Prisma migrations
npx prisma migrate dev

# Install bcryptjs
npm i bcryptjs && npm i @types/bcryptjs -D

# Install fakerjs
npm i @faker-js/faker -D
```
